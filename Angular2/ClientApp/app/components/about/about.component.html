Discworld Disorganiser

Welcome to the Discworld Disorganiser.

This application is designed to be a book and character search engine interface for the Discworld series of books.

To use this application, click on either 'Books' or 'Characters', these will take you to a search screen where you can search for a book or character within the Discworld series

Technology Used

This application runs on .NET Core and Angular2, with TypeScript compiled client-side scripting
